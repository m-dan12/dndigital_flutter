# 📝 Система сохранения заметок

## Что было добавлено

Реализована полноценная система сохранения и загрузки заметок на устройство с использованием локальной БД **Hive**.

### ✅ Функции

- **Автосохранение**: Заметки автоматически сохраняются с дебаунсингом (2 сек после последнего изменения)
- **Локальное хранилище**: Данные сохраняются в Hive БД
- **Синхронизация**: Контроллеры автоматически синхронизируются с состоянием
- **Индикатор статуса**: Показывает, когда идёт сохранение или произошла ошибка

## Как это работает

1. **Инициализация** (main.dart):
   - Инициализируется Hive
   - Регистрируется адаптер NoteHiveModel
   - Создается ViewModel с use cases

2. **При редактировании**:
   - Пользователь вводит текст
   - `NoteEditorViewModel` обновляет состояние
   - Срабатывает дебаунсированное автосохранение
   - Данные сохраняются в локальную БД

3. **При загрузке заметки**:
   ```dart
   NoteEditor(noteId: 'some-note-id')
   ```
   - ViewModel загружает заметку из БД
   - Контроллеры синхронизируются с данными

## Генерация Hive адаптера

Перед запуском приложения нужно сгенерировать адаптер:

```bash
dart pub get
dart run build_runner build
```

Это создаст файл `note_hive_model.g.dart` с автоматическим кодом сериализации.

## Использование в коде

### Создание новой заметки
```dart
NoteEditor()  // Будет создана новая заметка с уникальным ID
```

### Загрузка существующей заметки
```dart
NoteEditor(noteId: savedNoteId)  // Загружает заметку из БД
```

### Доступ к ViewModel
```dart
final viewModel = context.read<NoteEditorViewModel>();
final currentNote = viewModel.currentNote;
```

### Явное сохранение
```dart
await viewModel.saveNow();  // Сохранить без дебаунса
```
